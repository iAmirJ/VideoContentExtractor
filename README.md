## 📝 Project Description
Built an AI platform to automate a media client's content workflow using Python, FastAPI, and LLMs.

## ✨ Key Features
- **🤖 AI Summarizer:** Extracts precise insights from long videos.
- **✂️ Clip Generator:** Auto-generates engaging short clips for social media.
- **📄 Multi-Export:** Converts videos into PPT, PDF, and Word docs.
- **💬 Interactive Chat:** AI answers queries with clickable timestamps for instant video navigation.

## 🚀 Impact
Eliminated repetitive manual tasks and drastically scaled content production.

## 🛠️ Skills & Technologies
- Artificial Intelligence
- Large Language Models (LLMs)
- Python
- FastAPI
- Retrieval Augmented Generation (RAG)
